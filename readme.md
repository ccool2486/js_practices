# 設定

 git clone
```sh 
git clone https://github.com/ccool2486/js_practices.git
```

npm install
```sh 
npm install
```

# 常用指令

查看目前git狀態
```sh 
git status 
```

從github pull 資料回來
```sh 
git pull
```

提交所有變更
```sh 
git add -A 
git commit -m "[commit notes]"
```

Push到github
```sh 
git push origin master
```
